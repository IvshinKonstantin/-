# Программы на C#

В этом репозитории собраны несколько консольных программ на C#, каждая из которых демонстрирует работу с функциями, пользовательским вводом и базовыми операциями.

## Список программ

1. **Проверка положительности числа**  
   Функция `isPositive` проверяет, является ли число положительным.

2. **Сумма двух последних цифр числа**  
   Функция `sumLastNums` возвращает сумму двух последних цифр целого числа.

3. **Проверка заглавной буквы**  
   Функция `isUpperCase` определяет, является ли символ заглавной буквой (русской или английской).

4. **Проверка делимости чисел**  
   Функция `isDivisor` проверяет, делится ли одно число на другое без остатка.

5. **Сумма последних цифр двух чисел**  
   Функция `lastNumSum` возвращает сумму последних цифр двух целых чисел.

6. **Безопасное деление**  
   Функция `safeDiv` выполняет деление двух чисел, возвращая 0 в случае деления на ноль.

## Как использовать

1. Убедитесь, что у вас установлен [.NET SDK](https://dotnet.microsoft.com/download).
2. Скопируйте код нужной программы в файл `Program.cs`.
3. Откройте терминал в папке с файлом.
4. Выполните команды:
   ```bash
   dotnet new console
   dotnet run
